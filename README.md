# COVID19-PowerBI-Dashboard

## Overview
This repository contains a **Power BI dashboard** that visualizes COVID-19 data and provides insights into the pandemic’s impact globally. The dashboard includes key metrics such as total cases, total deaths, death rate, vaccination rates, fully vaccinated population, and other relevant measures. The project was developed as part of an **internship project**, using publicly available data from **Kaggle** and a Microsoft Power BI template provided during the program.

The dashboard demonstrates **data preprocessing, visualization, and interactive analytics** using Power BI.

---

## Dataset
- **Source:** [Kaggle COVID-19 Datasets](https://www.kaggle.com/datasets)  
- The dataset includes historical COVID-19 case counts, death counts, and vaccination statistics across countries.  
- Null values were replaced with 0, and decimal numbers were converted to whole numbers for clarity.  
- Only relevant columns and measures were included to create key insights.

---

## Key Measures & Visuals
- **Page 1 (Summary Metrics):**
  - Total Cases
  - Total Deaths
  - Death Rate
- **Page 2 (Vaccination Insights):**
  - Vaccination Rate
  - Maximum Total Vaccinations
  - Fully Vaccinated Population
  - People Vaccinated
- **Visualizations Used:**
  - Map Visual (Global Impact)
  - Line Chart (Trends Over Time)
  - Stacked Bar Chart
  - Cards (Key Metrics using DAX measures)
  - Text boxes for context and highlights

---

## System Development & Approach
- **Tool Used:** Microsoft Power BI  
- **Data Preprocessing:**
  - Handling null values
  - Converting decimals to whole numbers
  - Cleaning and organizing data for accurate visualization
- **Visualization Strategy:**
  - Focus on actionable metrics and clear visual storytelling
  - Use of interactive visuals to provide quick insights at a glance

---

## Results
- The dashboard provides a **clear, interactive summary of global COVID-19 data**.  
- Users can quickly analyze trends in cases, deaths, and vaccinations by country or region.  
- Key insights are highlighted through cards, maps, and charts to make the dashboard **easy to interpret**.
